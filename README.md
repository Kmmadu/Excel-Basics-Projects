# Excel Practice Lab

## Repository Overview
This repository is dedicated to practicing and improving Excel skills, particularly for data analysis and visualization. It contains various workbooks and projects that demonstrate essential Excel techniques, from basic functions to advanced data processing.

## Contents
- **Practice Workbooks**: Each workbook covers specific topics or functions in Excel.
- **Data Analysis Projects**: Small projects focused on analyzing sample datasets.
- **Visualization**: Examples of data visualizations created with Excel's charting tools.
  
## Skills Covered
- Basic Functions (SUM, AVERAGE, etc.)
- Conditional Formatting
- Data Validation
- Pivot Tables
- VLOOKUP and HLOOKUP
- Data Visualization
- Basic Data Cleaning
- IF Statements and Logical Functions
- ...and more as practice progresses!

## How to Use
Each file in this repository is a standalone Excel workbook. You can download them individually and open them with Excel to explore the functions and techniques demonstrated.
